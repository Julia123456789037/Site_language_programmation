meta-lique

Vue en cours

Balise : </meta>

Ce met dans le head, n'est pas visible par l'utilisateur. Il y a un meta pour le(s) auteur(s) du code html.

Exemples d'utilisations :

  • <meta charset="UTF-8" >

  • Définition de la langue et du nom de l'auteur :
    <meta name="Author" lang="fr" content="Justine BONDU" />
    (Plusieurs auteurs possibles séparés par des virgules.)

  • Définition de mot-clés qui caractérisent le mieux le site :
    <meta name="Keywords" lang="fr" content="Mots clés, espacés par une virgule" />

  • Catégorie de la page (pour les moteurs de recherche) :
    <meta name="Category" content="Création de site WEB" />

  • On peut spécifier le copyright de la page :
    <meta name="Copyright" content="Prénom NOM" />

  • On peut définir une description :
    C'est une zone fréquentée par les moteurs de recherche qui indexent le site,elle est utilisée comme résumé,
    <meta name="Description" content="La description de la page WEB" />

  • La langue de la page peut être définie sous la forme de 2 lettres : fr, en, de, es, it, ...
    <meta name="Language" content="fr" />

  • On peut interdire l’enregistrement de la page dans la mémoire cache :
    <meta http-equiv="Pragma" content="no-cache" />

  • On peut demander la réactualisation de la page toutes les X sec :
    <meta http-equiv="refresh" content="X" />

  • On peut aussi demander la redirection vers une autre page au bout de X secondes :
    <meta http-equiv="refresh" content="X;url=http://web.extension" />

ancrage sur terre


Vue en cours

Fait de lien clik-able.
Le lien permet d'aller :

  • ailleurs sur Internet, on doit alors spécifier une adresse complète ;
  • ailleurs sur le site local, on peut employer une adresse relative ;
  • à un endroit spécifique d'une page, on doit alors spécifier un point d'ancrage particulier.

Exemples d'utilisations :

<a href="liens_que_tu_veux" title="met_un_titre" > teste_qui_s_affiche </a>

local:
<a href="#ancre30" title="ancrage sur terre" > ancrage sur terre </a>

site externe:
<a href="https://validator.w3.org/" title="validator" > validator </a>

sur le meme site mais une page diff:
<a href="home.html" title="HOME" > Home </a>
<a href="home.html" title="HOME" > Home </a>
