Ou l'écrire


Ou écrire son code html, le mtitleieux que j'ai testé est visual studio code. Mais on peut très bien utilisé notepad++.



Structuration

N'a pas besoin de de titre ou autre il faut juste mettre directement.
On peut mettre le nom d'un balise, cela vas donc affecter toute la balise peut importe ou elle est dans le code html.
On peut mettre le nom d'une class, cela vas donc affecter toute la class peut importe ou elle est dans le code html.
On peut mettre le nom d'un id, cela vas donc affecter l'id peut importe ou elle est dans le code html.

L'id est unique.

Exemples d'utilisations :

hr{

margin-top: 2%;
width:75%;
color:#339c87;
}

.exemple{
text-decoration: underline;
font-size: 18px;
}

.pointsomm{
color:rgb(132, 193, 218);
border: 8px solid rgb(46, 1, 1);
background-color: #b12548;
opacity: 0.9;
margin-top: 25px;
margin-left: 25px;
width: 75%;
border-radius: 5px;
padding: 0px 50px 50px 50px;
}



coloration

Permet de changer la couleur du texte. Voir tableau ou chef d'oeuvre de HTML

Exemples d'utilisations :

p {
	color:white;
}
p {
	color: #ffffff;
}
p {
	color: rgb(255, 255, 255);
}
p {
	color: rgb(127, 11, 236);
}
p {
	color: rgb(200, 130, 233);
}
p {
	color: rgb(46, 1, 1);
}
